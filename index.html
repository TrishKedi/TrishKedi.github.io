About Me

I’m an AI Engineer currently pursuing a Master's in Artificial Intelligence for Smart Sensors and Actuators at Deggendorf Institute of Technology. My work blends machine learning, LLMs, and automation to create tools that improve workflows, solve real-world problems, and power intelligent systems. I enjoy building with LangChain, Hugging Face, FastAPI, Streamlit, and Python.

Projects

  • Personal Task Automation Agent

    An autonomous agent that breaks down high-level tasks like weekly planning using calendar APIs, RAG with personal notes, and vector memory. Built with LangChain, Ollama, Pinecone/FAISS, Streamlit, and Python.

  • Explain My Data – AI Data Profiler

    Upload a dataset, and the tool explains distributions, correlations, and outliers using an LLM. Visualizes insights using Matplotlib or Plotly. Built with LangChain, Pandas, and Streamlit.

  • AI-Powered Document Search

    Upload documents (PDF, DOCX, TXT) and ask questions via an LLM-powered search. Uses LangChain, FAISS/ChromaDB, Hugging Face, and Gradio.

  • Face Recognition Attendance System

    Uses OpenCV + FaceNet/Dlib to detect and track student attendance, with a React.js interface and PostgreSQL/MongoDB backend.

  • AI-Powered Medical Assistant

    Fine-tuned LLM to answer medical questions, explain symptoms, and summarize clinical documents based on trusted sources.

  • Persona Chatbot

    Custom LLM fine-tuned on curated data to mimic niche experts or public figures like Elon Musk. Combines scraping, prompt tuning, and model customization.

  • AI-Powered Code Debugger & Documentation Generator

    Analyzes buggy code, suggests fixes, and generates documentation using LLMs. Built with FastAPI, React, OpenAI API, Docker, and AWS.
